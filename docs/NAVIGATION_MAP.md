## Sistema de Desbloqueo

### Progresión por Actos

| Acto | Reinos Accesibles | Requisito |
|------|-------------------|-----------|
| I | Aether, Cosmos, Chaos, Logos | Inicio |
| II | + Atomos, Terra, Machina | 3 Eigenvalores |
| III | + Alchemy, Bios, Psyche | 6 Eigenvalores + 1 Conexión |
| IV | Todos + simulaciones ocultas | 10 Eigenvalores + 3 Conexiones |

### Conexiones Entre Reinos

Las conexiones se descubren cuando el jugador identifica relaciones entre fenómenos:

| Conexión | Reinos | Ejemplo |
|----------|--------|---------|
| Resonancia Gravitacional | Cosmos ↔ Aether | Las órbitas crean patrones rítmicos |
| Atractores Musicales | Chaos ↔ Aether | El Lorenz attractor tiene frecuencias |
| Simetría Infinita | Logos ↔ Cosmos | Los fractales aparecen en galaxias |
| Ondas Universales | Atomos ↔ Aether | Fourier conecta sonido y física |
| Tiempo Geológico | Terra ↔ Logos | Estratos siguen patrones matemáticos |
| Vida Emergente | Bios ↔ Chaos | Conway's Game of Life ↔ Evolución |
| Mente Artificial | Psyche ↔ Machina | Perceptrón ↔ Neurona |
| Transmutación | Alchemy ↔ Atomos | Reacciones ↔ Física de partículas |

## Navegación Técnica

### Transiciones

```javascript
// Desde AetherHub
enterPortal('cosmos') {
    this.cameras.main.fadeOut(500, 0, 0, 0);
    this.scene.start('CosmosRealm', {
        from: 'aether',
        playerState: this.player.getState()
    });
}

// Desde un Reino
enterSimulation('formacion-galaxias') {
    this.scene.launch('SimulationScene', {
        simulation: 'formacion-galaxias',
        realm: 'cosmos',
        objective: {...}
    });
}
```

### URLs de Simulaciones

```javascript
const SIMULATION_PATHS = {
    'formacion-galaxias': '/EigenLab/Astronomy/Astronomy Visual Lab/formacion-galaxias.html',
    'lorenz-attractor': '/EigenLab/...',
    // ...
};
```

### Integración con EigenLab

Opciones de integración:

1. **Iframe embebido** (más simple)
   - Cargar simulación en iframe
   - Comunicación via postMessage

2. **Integración directa** (más control)
   - Importar código de simulación
   - Compartir canvas/contexto

3. **Híbrido** (recomendado)
   - Algunas sims en iframe
   - Sims clave integradas para gameplay
